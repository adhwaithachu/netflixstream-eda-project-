## Netflix Streaming Data Analysis

This project explores and analyzes the Netflix streaming dataset (`netflixstream.csv`) to uncover patterns and insights about the content available on Netflix.  
It focuses on data cleaning, visualization, and storytelling using Python libraries like `pandas`, `matplotlib`, and `seaborn`.

---

# üìÅ Dataset Overview

The dataset `netflixstream.csv` contains information about TV shows and movies available on Netflix, including details such as title, director, cast, release year, and country of production.

### **Columns Description**

| Column Name | Description |
|--------------|-------------|
| `show_id` | Unique ID for each Netflix title |
| `type` | Type of content ‚Äî Movie or TV Show |
| `title` | Name of the show/movie |
| `director` | Director of the title (if available) |
| `cast` | Main actors in the title |
| `country` | Country where the title was produced |
| `date_added` | Date when the title was added to Netflix |
| `release_year` | Year of release |
| `rating` | Content rating (e.g., PG-13, TV-MA) |
| `duration` | Duration in minutes or number of seasons |
| `listed_in` | Categories/Genres |
| `description` | Short summary of the title |

- The USA and India are among the top content-producing countries.

- Noticeable growth in releases after 2015.

- Common genres include International Movies, Dramas, and Comedies.
